## can't find remote repository <repository>

- 원인: credential error - config에 저장된 id 값이 해당 원격저장소를 볼 수 있는 권한이 없음
- 해결: 
  - git config에 등록된 id/pw 계정 정보가 입력한 원격저장소에 접근 권한이 있는지 확인한다.
    - 접근 권한을 얻던가, 아니면 git config에 저장된 계정 정보를 수정한다.
